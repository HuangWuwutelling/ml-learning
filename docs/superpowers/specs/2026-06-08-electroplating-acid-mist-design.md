# 电镀厂酸雾：从工艺产污到大气扩散与沉降 — 设计文档

## 概述

环境模型系列第 07 篇。以典型镀铬生产线为场景，沿「工艺产污 → 集气收集 → 碱液洗涤 → 烟囱排放 → 高斯烟羽扩散 → 干湿沉降 → 土壤酸化」全链条建模，展示环境工程中源强核算与大气扩散的完整方法。

姊妹篇（08 电镀厂 Cr(VI)）共用大气扩散底盘，但下垫面过程不同（Cr 累积与健康风险）。

## 文章标题（暂拟）

**电镀厂酸雾飘多远？从工艺产污到沉降的全程模拟**

## 模型架构

### 六步质量平衡链条

```
产污量 D (g/h)
  × 集气效率 η_coll = 90%
  × 洗涤效率 η_scrub (铬酸雾 95%, 硫酸雾 90%, HCl 95%)
  = 烟囱排放量 Q_stack (g/s)
  → 高斯烟羽扩散 → 地面浓度 C(x,y,0)
  → 干湿沉降 F_dry + F_wet → H⁺ 当量 → 土壤酸化比 R
```

### 参数来源

所有核心参数来自两项国家标准原文（已通过网络核实）：

| 标准 | 关键内容 |
|------|---------|
| **GB 21900-2008**《电镀污染物排放标准》 | 表5 大气排放限值（铬酸雾 0.05 mg/m³, 硫酸雾 30 mg/m³, HCl 30 mg/m³）；表6 基准排气量 |
| **HJ 984-2018**《污染源源强核算技术指南 电镀》 | 附录B 产污系数表（铬酸雾 0.38 g/m²·h, 硫酸雾 25.2 g/m²·h）；附录F 治理效率 |

### 第一步：产污源强

采用产污系数法，公式 D = G_s × A（HJ 984-2018 式 1）：

| 工序 | 产污系数 | 产生量 (A=2m²) |
|------|---------|----------------|
| 镀铬 → 铬酸雾 | 0.38 g/m²·h \[HJ 984 表B.1\] | 0.76 g/h |
| 酸洗 → 硫酸雾 | 25.2 g/m²·h \[HJ 984 表B.1\] | 50.4 g/h |
| 酸洗 → 氯化氢 | 220 g/m²·h (取中值) \[HJ 984 表B.1\] | 440 g/h |

### 第二步：槽边集气

集气效率 η_coll = 90%（设计规范典型值）。未收集的 10% 作为无组织逸散定性讨论。

### 第三步：湿式洗涤塔

| 污染物 | 去除效率 | 来源 |
|--------|---------|------|
| 铬酸雾喷淋回收 | ≥95% | HJ 984-2018 附录F 表F.1 |
| 硫酸雾碱液喷淋 | ≥90% | HJ 984-2018 附录F 表F.1 |
| 氯化氢碱液喷淋 | ≥95% | HJ 984-2018 附录F 表F.1 |

核算排放浓度（以铬酸雾为例）：
- Q_stack = 0.76 × 0.9 × (1-0.95) = 0.034 g/h
- 排气量 = 74.4 m³/m² × 2 m² = 148.8 m³/h（GB 21900 表6）
- 排放浓度 = 0.034 / 148.8 = 0.23 mg/m³
- GB 21900 限值 = 0.05 mg/m³ → 不达标！

**结论：单级洗涤难以达标，需二级处理或强化碱液喷淋。这是文章的重要论点。**

### 第四步：高斯烟羽扩散

沿用 02 篇高斯烟羽模型，核心公式：

```
C(x,y,0) = Q/(π·u·σ_y·σ_z) × exp(-y²/(2σ_y²)) × exp(-H²/(2σ_z²))
```

参数：
- Q：各污染物烟囱排放量 (g/s)，来自前三步
- u = 3 m/s（年均风速，D 类稳定度）
- H = 20 m（物理 15m + 抬升 5m，Briggs 公式）
- σ_y, σ_z：扩散参数（Power-law，D 类稳定度）

### 第五步：干湿沉降

**干沉降：** F_dry = v_d × C(x,y,0)
- 铬酸雾气溶胶 v_d = 0.3~0.5 cm/s
- 硫酸雾气溶胶 v_d = 1.0~2.0 cm/s

**湿沉降：** Λ = 10⁻⁴ s⁻¹（气溶胶清除系数，Seinfeld & Pandis）

总沉降换算为 H⁺ 当量：H₂SO₄ → 2H⁺, HCl → 1H⁺, H₂CrO₄ → 2H⁺

### 第六步：土壤酸化评估

酸化风险比 R = H⁺ 年沉降量 / 土壤酸缓冲容量

| R 值 | 风险等级 |
|------|---------|
| < 0.5 | 安全 |
| 0.5~1.0 | 潜在酸化风险 |
| ≥ 1.0 | 高风险 |

## 模型代码设计

**文件：** `models/electroplating_acid_mist.py`

采用 standalone 脚本风格（与 `atmo_cd_deposition.py` 一致），包含：

1. `calc_emission_rate()` — 前三步：产污 → 集气 → 洗涤 → 烟囱源强
2. `gaussian_plume()` — 高斯扩散计算
3. `calc_deposition()` — 干湿沉降通量
4. `calc_acidification_risk()` — 土壤酸化评估
5. `main()` — 生成 5 张图

### 参数标注约定（沿用已有模式）

- `[S]` = 标准值（GB/HJ）
- `[L]` = 文献值
- `[E]` = 估算值（注明依据）
- `[C]` = 计算值
